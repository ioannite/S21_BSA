Wireframes системы онлайн-записи в барбершоп

Обзор

Данный проект содержит интерактивные wireframes (прототипы) для системы онлайн-записи на стрижку в барбершоп. Wireframes реализованы как одностраничное приложение (SPA) с использованием HTML5, CSS3 и JavaScript, следуя принципам mobile-first дизайна.

Цель

Разработать пользовательские интерфейсы для:

Клиентов (регистрация, выбор услуги, мастера, времени, подтверждение записи)

Менеджеров (управление расписанием, мастерами, просмотр записей и отчетов)

Мастеров (просмотр расписания, клиентов, отзывов)


Структура файлов
